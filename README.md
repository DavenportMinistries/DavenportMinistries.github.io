# DavenportMinistries.github.io

Public website for Davenport Ministries LLC — [davenportministries.com](https://davenportministries.com)

Built with Jekyll, hosted on GitHub Pages.

## Local development

```bash
gem install bundler
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000.

## Adding a project

Edit the `#projects` section in `index.html` — copy an existing `.project-card` block and update the name, description, URL, and icon emoji.
